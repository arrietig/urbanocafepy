# Urbano Café

Sitio web de Urbano Café — café de especialidad tostado en Asunción, Paraguay.

## Vista previa local

```bash
python3 -m http.server 8765
```

Abrir http://localhost:8765

## Stack

- HTML estático
- CSS vanilla (sin frameworks)
- JavaScript vanilla
- Google Fonts: Archivo Black, Cormorant Garamond, Inter

## Estructura

```
.
├── index.html      # Página principal
├── styles.css      # Estilos
└── script.js       # Interacciones (menú, scroll reveal, carrito)
```
